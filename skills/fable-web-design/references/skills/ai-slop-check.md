# AI Slop Check: Reject Generic Tropes

Run this check on every design before delivery. These patterns are the
signature of AI-generated design — recognizable, overused, and quality-lowering.

## Checklist (all must pass)

- [ ] No generic gradient backgrounds (purple-to-blue, sunset, aurora)
- [ ] No emoji used as decoration or illustration
- [ ] No rounded corners + left-border accent as default card pattern
- [ ] No hand-drawn SVG illustrations as filler
- [ ] No Inter, Roboto, Arial as default font choice without brand reason
- [ ] No editorial-warm house style (cream background + serif display +
      terracotta accent) unless brand calls for it
- [ ] No pure white (#fff) or pure black (#000) as primary surfaces
- [ ] No random invented colors
- [ ] No random spacing values (everything on a clear scale)
- [ ] No random decorative elements that don't communicate or structure

## Why This Matters

These tropes are recognizable precisely because they appear in every
AI-generated design. The goal isn't to avoid them forever — it's to avoid
them as defaults. If the design direction genuinely calls for a gradient or
Inter font, use it intentionally. The difference is deliberation vs. default.
