# Hierarchy and Rhythm Review

## Hierarchy
- **Primary differentiation**: size, color, weight, position
- **Secondary**: supporting information clearly subordinate
- **Tertiary**: meta info, timestamps, ancillary links
- **Density**: consistent — no sections feel cramped while others breathe
- **5-second test**: can someone identify the primary action within 5 seconds?

## Rhythm
- **Spacing scale discipline**: every gap maps to the scale; no ad-hoc values
- **Type scale discipline**: every font size maps to the scale; no ad-hoc px
- **Repetition**: consistent patterns for consistent elements
- **Strategic variation**: variation has a reason, not randomness
- **Color palette discipline**: colors used consistently for consistent meanings
- **Section structure**: predictable rhythm of content blocks
- **Alignment**: strong alignment grid; no floating elements without purpose
