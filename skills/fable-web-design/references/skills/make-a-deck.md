# Make a Deck: Presentation

Produce slide decks as HTML presentations. Each slide is a section in a
single HTML file with CSS for layout and JS for navigation.

## Structure

- Title slide: name, date, one-line description
- Content slides: one idea per slide, visual over text
- Closing slide: one takeaway

## Design Rules

- Consistent typography throughout
- One primary visual per slide (not a wall of text)
- High contrast for projectors
- Minimum 24pt body text
- Each slide readable in 30 seconds or less

## Output Format

Single HTML file with inline CSS and JS. Arrow-key navigation. Clean,
minimal, opinionated.
